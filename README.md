## üõ†Ô∏è Using sfx-rs

sfx-rs allows you to create self-extracting archives easily. Follow these steps to create your first archive:

1. Open your Command Prompt or Terminal.
2. Navigate to the folder where your files are located.
3. Run the following command:

   ```bash
   sfx-rs create your-archive-name.sfx file1.txt file2.txt
   ```

Replace `your-archive-name.sfx` with the desired name for your archive, and list the files you want to include.

4. Your self-extracting archive will be ready!

## ‚ùì FAQs

### What is a self-extracting archive?

A self-extracting archive is a compressed file that contains its own extraction program. This means users can run it without needing additional software to extract the files inside.

### Can I create archives on Linux?

Yes, sfx-rs works on Linux systems. Follow the installation instructions provided above.
